## RGB
RGB is controller on the first layer.
Backspace: RGB toggle.
Q: Decrease HUE
W: Increase HUE
E: Decrease SAT
R: Increase SAT
Arrow Up: Increase brightness
Arrow Down: Decrease brightness
Side Arrows: Select RGB effect
